CRM — Customer Relationship Management System

A modern and scalable CRM (Customer Relationship Management) System built to streamline customer interactions, lead management, sales tracking, support operations, and business communication. This system is designed with clean architecture, reusable modules, and database-driven workflows using PHP and MySQL.
